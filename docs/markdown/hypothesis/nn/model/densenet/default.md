Module hypothesis.nn.model.densenet.default
===========================================
DenseNet defaults.

Variables
---------

    
`batchnorm`
:   Default batch normalization flag in Hypothesis.

    
`bottleneck_factor`
:   Default bottleneck-factor in DenseNet architectures.

    
`channels`
:   Default number of data channels (e.g., channels in images).

    
`convolution_bias`
:   Add biases in convolutions by default in Hypothesis.

    
`depth`
:   Default DenseNet architecture (depth).
